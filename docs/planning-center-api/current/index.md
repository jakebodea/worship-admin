# Planning Center Current

The Current API requires no scopes and imposes no app access or permissions, so
any OAuth app can use it to recover whoami-level details about the current person
and their organization.

It's intended to be used through either `/current/v2/me` or
`/current/v2/me?include=organization`.



## App

- App slug: `current`
- Base docs endpoint: `https://api.planningcenteronline.com/current/v2/documentation`
